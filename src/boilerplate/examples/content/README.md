# Example Content and Templates

This directory contains comprehensive example content and templates for the JSON CMS Boilerplate system. These examples demonstrate best practices and provide starting points for different project types.

## Structure

- `pages/` - Sample page JSON files with components, SEO, and metadata
- `blocks/` - Example block definitions with variants and constraints
- `plugins/` - Sample plugin implementations
- `templates/` - Boilerplate templates for different project types
- `integrations/` - Common integration examples and patterns

## Usage

These examples are designed to:
1. Demonstrate the full capabilities of the JSON CMS system
2. Provide copy-paste starting points for new projects
3. Show best practices for content structure and organization
4. Illustrate integration patterns with common use cases

## Getting Started

1. Copy the relevant template from `templates/` to your project
2. Customize the example pages and blocks for your needs
3. Install any required plugins from `plugins/`
4. Follow the integration examples in `integrations/`

Each example includes detailed comments explaining the structure and purpose of each element.
