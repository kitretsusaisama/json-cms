## 🔧 **How to Use in Production**

### **1. Basic Integration**
```jsx
import { AdvancedCookieConsent } from './AdvancedCookieConsent';

function App() {
  return (
    <>
      {/* Your app content */}
      <AdvancedCookieConsent />
    </>
  );
}
```

### **2. Listen for Consent Events**
```javascript
useEffect(() => {
  const handleConsentUpdate = (event) => {
    const { categories, method, timestamp } = event.detail;
    
    // Load scripts based on consent
    if (categories.analytics) {
      loadGoogleAnalytics();
    }
    if (categories.marketing) {
      loadFacebookPixel();
      loadGoogleAds();
    }
  };

  window.addEventListener('cookieConsentUpdated', handleConsentUpdate);
  return () => window.removeEventListener('cookieConsentUpdated', handleConsentUpdate);
}, []);
```

### **3. Customize Categories**
The component includes 5 cookie categories:
- **Essential**: Required cookies (cannot be disabled)
- **Functional**: User experience enhancements
- **Analytics**: Usage tracking and performance
- **Marketing**: Advertising and personalization
- **Personalization**: Content customization
